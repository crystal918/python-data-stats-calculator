# Python Data Stats Calculator 📊

A dynamic command-line data analysis tool built with Python.

## Features
Accepts any amount of numbers, type done when finished. Calculates sum, average, minimum, maximum and range. Shows total count and provides insights based on the average. Handles invalid inputs and empty lists gracefully.

## Concepts Used
- Functions and lists
- Dynamic input with while loops
- Try/except error handling
- Defensive programming
- F-strings and conditional logic

## How to Run
Type: python data_stats.py in your terminal
